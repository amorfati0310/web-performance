3. 최근 로딩 성능의 기준은 사용자 입장에서 고려되고 있고 사용자에게 의미 있는 내용을 최대한 먼저 보여주자는 개념을 나타내는 용어를 쓰세요.

4 리소스 최적화 방법 중 틀린 것은?

6. Minifying 하여 용량을 줄인다.
